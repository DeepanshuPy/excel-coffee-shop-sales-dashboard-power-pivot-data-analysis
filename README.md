# ☕ Coffee Shop Sales Dashboard (Excel | Power Pivot | Data Analysis)

## 🔹 Objective
Analyze coffee shop sales data to uncover business insights such as peak hours, top-selling products, and overall performance using Excel and Power Pivot.

---

## 🛠️ Tools Used
- Microsoft Excel  
- Power Pivot  
- Pivot Tables  
- Data Visualization  

---

## 🔹 Key Steps Performed
- Created separate column for **product size (Lg, Sm, Rg)**  
- Calculated **Total Bill = Quantity × Unit Price**  
- Cleaned and adjusted **time column**  
- Built pivot tables based on business requirements  
- Created interactive charts (**bar, pie, line**)  
- Designed KPI metrics:
  - Total Sales  
  - Footfall  
  - Average Order Value (AOV)  
- Added slicers for **monthly and daily analysis**  

---

## 📊 Business Insights
- Peak hours are between **7 AM – 10 AM**  
- Coffee and tea are the most ordered items, contributing to **~67% of total sales**  
- Morning time drives the majority of revenue  

---

## ⚠️ Challenges Faced
- Formatting slicers and button colors (selected/unselected states)  
- Designing a clean and user-friendly dashboard layout  

---

## 🎯 Outcome
Developed an interactive dashboard that helps understand sales performance and supports better business decisions.
